# **App Name**: SkillSync Landing

## Core Features:

- Animated Navbar: Semi-transparent navbar with logo animation and micro-interactions on menu items using glassmorphism background and looping animations.
- Hero Section: Dynamic headline with typing animation, pulsing sub-headline, and primary CTA button with shimmering gradient and glow effects.
- Interactive Particle Network: 3D particle network model simulating skill matching between job seekers and companies, with glowing connections and pulse animation on CTA hover.
- AI Skill Analysis: Utilize a tool for AI to interpret and verify skill inputs, which organizes information into visualizable skill graph outputs
- Skill Graph Visualizer: Interactive 3D force-directed graph displaying user skills as nodes and relatedness as connecting lines, with sample profiles and smooth reconfiguration animations.
- Animated Stats: Number count-up animations for displaying verified skills and matches made, triggering on scroll.
- Final CTA Animation: Prominent call-to-action button featuring a particle effect and a more pronounced animation upon hover in the gradient mesh background

## Style Guidelines:

- Primary background: Deep Charcoal to Jet Black gradient (#0F0F0F to #000000).
- Secondary surfaces: Dark Gunmetal Gray (#1A1A1A) with subtle gradients.
- Accent color: Electric Cyan (#00EEFF) for CTAs, highlights, and key data points.
- Body text: 'Inter' sans-serif font for a clean and modern feel.
- Headlines: 'Source Code Pro' monospace font to reinforce the skill-based theme; use this also for code snippets, and statistics.
- Ensure all components are built using .jsx not .tsx and that Tailwind CSS framework is applied.
- Micro-interactions: Refined hover/focus states for interactive elements using scale, color, or glow transitions.
